- Introduction to Search
  - A* search algorithm
- Program A* Search
  - Algorithm will be broken down into several exercises for you 
  - Algorithm summary and structure of code for exercises
- C++ Features
  - Pass-by-reference
  - const and constexpr
  - Arrays
